# Manga Match! — Framtida idéer

## Spellägen
- **Time Attack** — Samla så många poäng som möjligt på 60/90 sekunder. Separat spelmod i menyn, inte default
- **Endless Mode** — Inget slut, brädet shufflar sig självt. Bara highscore
- **Versus / PvP** — Delad skärm eller online, combo:s skickar hinder till motståndaren

## Progression
- **Upplåsbara karaktärer/skins** — Belöning för stjärnor eller achievements
- **Karaktärsporträtt i menyn/level-picker** — En anime-karaktär som "värd" för spelet
- **World-teman** — Varje 10-pack nivåer har unikt visuellt tema (stad, skog, rymd)
- **Boss-banor** — Var 10:e nivå: specialregler, unik fiende-mekanik

## Visuellt
- **Halftone/screentone-mönster** — Klassisk manga-textur som bakgrund på paneler (CSS-pattern)
- **Speed lines vid combos** — Radiella linjer bakom brädet vid chain ≥2
- **Karaktärsreaktioner** — Porträtt som reagerar på bra/dåliga drag
- **Anime cut-in** — Kort animation vid stora combos (fullskärm slash-effekt)

## Ljud
- **Musik** — Royalty-free anime/lo-fi-loop, en per world-tema. Inte genererad med Web Audio
- **Röstklipp** — Korta japanska utrop vid combos ("Sugoi!", "Yatta!")

## Socialt / Platform
- **GameVolt.io-integration** — Online high-score, achievements
- **Dagliga tävlingar** — Samma seed för alla spelare, leaderboard
- **Delningsfunktion** — Screenshota resultat, dela på sociala medier

## Kvalitet
- **Offline-stöd (PWA)** — Service worker, installera som app
- **Tillgänglighet** — Färgblindsläge, tangentbordsstyrning
- **Statistik-sida** — Totala matcher, favorit-tile, längsta chain, speltid
